# Tech Job Market Analytics Pipeline

## Overview
An end-to-end Data Engineering project that collects real-time job postings using APIs, processes data through an ETL pipeline, stores it in PostgreSQL, and visualizes hiring trends using Power BI.

## Features
- Real-time Job Data Collection
- Automated ETL Pipeline
- PostgreSQL Data Warehouse
- SQL Analytics
- Power BI Dashboard
- Skill Demand Analysis
- Salary Trend Analysis
- Company Hiring Insights
- Location Analysis

## Tech Stack
- Python
- Pandas
- PostgreSQL
- SQL
- Power BI
- REST APIs
- Git & GitHub
